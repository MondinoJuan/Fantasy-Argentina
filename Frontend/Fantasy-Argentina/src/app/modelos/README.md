# Modelos Frontend

Se agregaron modelos para todas las entidades del backend:

- bid
- league
- match
- matchday
- matchdayMarket
- negotiation
- participant
- participantMatchdayPoints
- participantSquad
- playerClause
- playerPerformance
- playerPointsBreakdown
- realPlayer
- realTeam
- shielding
- tournament
- transaction
- user

Para cada entidad se crearon 5 archivos:

1. `add<Entity>.interface.ts`
2. `<entity>.interface.ts`
3. `<entity>.patch.interface.ts`
4. `response<Entity>.interface.ts`
5. `<entity>.collection.interface.ts`
